💈 Hair Day

🇧🇷 Sobre o Projeto

Este projeto é uma aplicação web de agendamento de horários para salão/barbearia. O usuário seleciona uma data, escolhe um horário disponível dividido por período (manhã, tarde e noite), informa o nome do cliente e confirma o agendamento.

Este repositório foi desenvolvido como parte dos meus estudos na Rocketseat, com foco em praticar JavaScript puro (vanilla), manipulação de datas, organização de projetos em módulos e integração com uma API simulada.

🚧 Projeto em desenvolvimento.

🇺🇸 About the Project

This project is a web scheduling application for a hair salon/barbershop. The user selects a date, chooses an available time slot divided by period (morning, afternoon, and evening), enters the client's name, and confirms the appointment.

This repository was developed as part of my studies at Rocketseat, focused on practicing vanilla JavaScript, date handling, modular project structure, and integration with a simulated API.

🚧 Project in progress.

🚀 Tecnologias | Technologies

🎯 Funcionalidades | Features

* 📅 Seleção de data com validação (não permite datas passadas) | Date selection with validation (prevents past dates)
* 🕐 Listagem de horários disponíveis por período | Available time slots listed by period
* ✅ Indicação visual de horários disponíveis/indisponíveis | Visual indication of available/unavailable time slots
* 🧑 Cadastro de cliente no agendamento | Client registration on scheduling
* 🔄 Atualização dinâmica da interface | Dynamic interface updates
